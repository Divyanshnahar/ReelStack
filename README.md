## ЁЯза What is ReelsStack?

**ReelsStack** is a full-stack AI SaaS application that automates the end-to-end creation of engaging short-form videos. From a simple topic idea, it generates scripts, creates AI images for each scene, produces professional voiceovers, auto-captions the audio, and renders a final `.mp4` video тАФ entirely in the cloud.

### тЬи What Users Can Do

| Capability | Description |
|---|---|
| ЁЯУЭ **Script Generation** | Generate structured scene scripts from a topic, style, and duration |
| ЁЯЦ╝я╕П **AI Image Creation** | Auto-generate cinematic 1080├Ч1920 images per scene |
| ЁЯОЩя╕П **AI Voiceover** | Convert narration into professional MP3 via ElevenLabs |
| ЁЯУД **Auto Captions** | Word-level synchronized captions via Deepgram |
| тШБя╕П **Cloud Rendering** | Render final video using Remotion + GitHub Actions |
| ЁЯЧВя╕П **Dashboard** | Manage, download, edit, and delete your videos |

## тЪб Core Features Deep Dive

### 1я╕ПтГг AI Script Generation

Users provide a **topic**, **style**, and **duration**. Gemini AI returns a structured JSON array:

```json
[
  {
    "imagePrompt": "Cinematic aerial view of a futuristic city at sunset",
    "contentText": "Welcome to the cities of tomorrow, where technology meets humanity."
  }
]
```

### 2я╕ПтГг AI Image Generation (Hugging Face SDXL)

- Each scene's `imagePrompt` is sent to Stable Diffusion XL
- Images generated in **1080├Ч1920** portrait format
- Uploaded to Cloudinary and stored securely

### 3я╕ПтГг AI Voiceover (ElevenLabs)

- All scene `contentText` fields are combined
- Converted to MP3 using ElevenLabs' professional voice models
- Uploaded to Cloudinary for cloud storage

### 4я╕ПтГг Auto Captions (Deepgram)

- Cloudinary audio URL sent to Deepgram
- **Word-level timestamps** extracted
- Captions synchronized frame-by-frame in Remotion

### 5я╕ПтГг Cloud Video Rendering (Remotion + GitHub Actions)

- Video metadata saved to PostgreSQL
- GitHub Actions workflow triggered automatically
- Remotion composes final video with images, audio, and captions
- Final `.mp4` uploaded as a build artifact

### 6я╕ПтГг User Dashboard

Users can **view**, **edit titles/descriptions**, **download**, and **delete** their videos тАФ all from a clean, unified dashboard.

## ЁЯФз Local Setup

### Prerequisites

- Node.js 18+
- PostgreSQL (or a free [Neon DB](https://neon.tech/) instance)
- Accounts on: Clerk, Hugging Face, ElevenLabs, Deepgram, Cloudinary, Google AI Studio

---

### 1я╕ПтГг Clone the Repository

```bash
git clone https://github.com/your-username/ReelsStack.git
cd ReelsStack
```

### 2я╕ПтГг Install Dependencies

```bash
npm install
```

### 3я╕ПтГг Configure Environment Variables

Create a `.env.local` file in the root directory:

```env
# Clerk Authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

# Database
DATABASE_URL=
NEXT_PUBLIC_DATABASE_URL=

# AI Services
HUGGING_FACE_API_KEY=
ELEVENLABS_API_KEY=
NEXT_PUBLIC_GEMINI_API_KEY=
DEEPGRAM_API_KEY=

# Cloudinary
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

### 4я╕ПтГг Run Database Migrations

```bash
npx drizzle-kit generate
npx drizzle-kit migrate
```

### 5я╕ПтГг Start the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
